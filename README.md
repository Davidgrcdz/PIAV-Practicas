# 👁️‍🗨️ Procesamiento de Imágenes y Audio/Vídeo (PIAV)

Bienvenido a mi repositorio de **PIAV**. En este espacio recojo todos los proyectos, ejercicios y *notebooks* desarrollados para aprender y aplicar técnicas de visión artificial, procesamiento de señales y análisis multimedia.

## 🚀 Contenido del Repositorio

El proyecto está estructurado por prácticas, avanzando desde el procesamiento básico de señales e imágenes hasta la extracción de características complejas:

### 1. 🖼️ Práctica 1: Fundamentos y Filtrado de Imágenes
[`/Practica1-Filtros_Basicos`](./Practica1-Filtros_Basicos)
* **Descripción:** Introducción al manejo de imágenes digitales. Aplicación de transformaciones básicas, histogramas, y uso de filtros espaciales (suavizado, realce de bordes) para mejorar la calidad o extraer información inicial.

### 2. ✂️ Práctica 2: Segmentación y Detección
[`/Practica2-Segmentacion`](./Practica2-Segmentacion)
* **Descripción:** Técnicas para separar los objetos de interés del fondo. Incluye métodos de umbralización (Otsu), detección de contornos (Canny) y operaciones morfológicas (dilatación, erosión).

### 3. 🎵 Práctica 3: Procesamiento de Audio
[`/Practica3-Audio_Señales`](./Practica3-Audio_Señales)
* **Descripción:** Salto al mundo unidimensional. Análisis de señales de audio, paso al dominio de la frecuencia mediante la Transformada de Fourier (FFT), y extracción de características sonoras (como MFCCs).

### 4. 🔍 Práctica 4: Extracción de Características y Matching
[`/Practica4-Features_Matching`](./Practica4-Features_Matching)
* **Descripción:** Uso de descriptores avanzados (como SIFT, SURF o ORB) para encontrar puntos clave en imágenes y lograr emparejarlos (*matching*) entre diferentes fotografías, base fundamental para el reconocimiento de objetos o la creación de panorámicas.

## 🛠️ Stack Tecnológico
Todo el código está desarrollado para ser fácil de seguir y replicar. Las herramientas principales utilizadas en este repositorio son:
* **Python** como lenguaje principal.
* **OpenCV (`cv2`)** y **Scikit-Image**: Para todo el procesamiento de visión artificial.
* **Librosa / SciPy**: Para el análisis y tratamiento de señales de audio.
* **Matplotlib / Seaborn**: Para la visualización de imágenes, espectrogramas y gráficas de resultados.
* **Jupyter Notebook**: Como entorno interactivo de desarrollo.
