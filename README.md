# 👁️‍🗨️ Multimedia Processing (Imagen, Audio y Vídeo)

Bienvenido a mi repositorio de **Procesamiento Multimedia**. En este espacio recojo todos los proyectos, ejercicios y *notebooks* desarrollados para aprender y aplicar técnicas avanzadas de visión artificial, procesamiento de señales acústicas y análisis de vídeo.

## 🚀 Contenido del Repositorio

El proyecto está estructurado de manera progresiva a través de 7 prácticas principales y una entrega especial, avanzando desde transformaciones básicas hasta el análisis complejo de profundidad y movimiento:

### 1. 🖼️ Práctica 1: Fundamentos y Filtrado de Imágenes
[`/Practica 1`](./Practica%201)
* **Descripción:** Introducción al manejo de imágenes digitales. Aplicación de transformaciones básicas, ecualización de histogramas, y uso de filtros espaciales (suavizado, paso bajo/alto, realce de bordes) para mejorar la calidad y extraer información inicial.

### 2. ✂️ Práctica 2: Segmentación y Detección
[`/Practica 2`](./Practica%202)
* **Descripción:** Técnicas para separar los objetos de interés del fondo. Incluye métodos de umbralización, detección de contornos, y operaciones morfológicas (dilatación, erosión, apertura y cierre).

### 3. 🎵 Práctica 3: Procesamiento y Análisis de Audio
[`/Practica 3`](./Practica%203)
* **Descripción:** Salto al mundo unidimensional de las señales acústicas. Análisis de audio, transformación al dominio de la frecuencia mediante la Transformada de Fourier (FFT), y extracción de características sonoras fundamentales.

### 4. 🔍 Práctica 4: Extracción de Características y Matching
[`/Practica 4`](./Practica%204)
* **Descripción:** Detección de puntos clave y descriptores locales en imágenes. Uso de algoritmos para lograr emparejarlos (*feature matching*) entre diferentes fotografías, base para el reconocimiento de objetos o la creación de imágenes panorámicas.

### 5. 🚗 Práctica 5: Procesamiento Avanzado
[`/Practica 5`](./Practica%205)
* **Descripción:** Aplicación de algoritmos de nivel intermedio-avanzado en visión por computador, explorando transformaciones geométricas complejas o detección de patrones específicos en la imagen.

### 6. 📹 Práctica 6: Análisis de Vídeo y Flujo Óptico
[`/Practica 6`](./Practica%206)
* **Descripción:** Procesamiento de secuencias temporales (vídeo). Detección de movimiento, sustracción de fondo y cálculo del flujo óptico (*Optical Flow*) para rastrear el desplazamiento de píxeles entre *frames*.

### 7. 🎯 Práctica 7: Seguimiento y Aplicaciones
[`/Practica 7`](./Practica%207)
* **Descripción:** Implementación de algoritmos de seguimiento de objetos (*Object Tracking*) a lo largo de un vídeo, combinando técnicas de las prácticas anteriores para mantener la identidad de un objeto en movimiento.

## 🛠️ Stack Tecnológico
Todo el código está desarrollado para ser fácil de seguir y replicar. Las herramientas principales utilizadas en este repositorio son:
* **Python** como lenguaje principal.
* **OpenCV (`cv2`)** y **Scikit-Image**: Para todo el procesamiento de visión artificial y vídeo.
* **Librosa / SciPy**: Para el análisis y tratamiento de señales de audio.
* **Matplotlib / Seaborn**: Para la visualización de imágenes, espectrogramas y gráficas.
* **Jupyter Notebook**: Como entorno interactivo de desarrollo (`.ipynb`).
